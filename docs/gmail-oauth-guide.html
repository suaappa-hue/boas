1. ⚙ OAuth 2.0 configuration 클릭
2. Use your own OAuth credentials 체크
3. 위에서 복사한 Client ID와 Client Secret 입력

Step 1: Select & authorize APIs

좌측 목록에서 Gmail API v1 찾기 → 아래 scope 선택:

https://mail.google.com/

Exchange authorization code for tokens 클릭
