블랙리스트 기능 — Airtable 설정

아래 2가지만 Airtable UI에서 만들어주세요. 작업 시간 약 3분.

1

"블랙리스트" 새 테이블 생성

Base: appvXvzEaBRCvmTyU (BOAS 메인 베이스) 에서 좌측 상단의 "+" 버튼으로 새 테이블 생성. 이름은 정확히 블랙리스트.

필드 구성 (정확한 이름 + 타입으로)

체크 필드명 타입 설명
연락처 Single line text ★ Primary 정규화된 형태(예: 01077556700). 첫 번째 필드 이름을 "연락처"로 변경.
사유 Long text 차단 사유 메모. 선택 입력.
등록일시 Date and time Include time = ON, Time format = 24시간, Time zone = Asia/Seoul.
등록자 Single line text 관리자 이메일 또는 'system'. 선택.
차단횟수 Number Format = Integer, Default = 0. Worker가 매치마다 +1.
마지막차단일시 Date and time 등록일시와 동일 설정.

⚠️ 필드명에 오타 있으면 Worker/API가 매치 못합니다. 위 표의 한글 이름 그대로 입력하세요.

2

"고객접수" 테이블 — 상태 옵션에 '스팸' 추가

  1. 고객접수 테이블 열기
  2. 상태 컬럼 헤더 우클릭 → Edit field
  3. 옵션 목록 맨 아래에 스팸 추가 (색상: 빨강 권장)
  4. Save
✏️ 기존 옵션: 신규 / 대기 / 상담중 / 진행중 / 완료 → 여기에 스팸 한 줄 추가만 하면 됩니다.
3

완료 후 확인

  • ✓ "블랙리스트" 테이블이 사이드바에 보임
  • ✓ 6개 필드 모두 표 그대로 생성됨
  • ✓ "고객접수" 상태 옵션 펼치면 '스팸'이 보임

위 3가지 확인되면 "완료"라고 알려주세요.

제가 나머지 코드(API, 페이지, Worker)를 마저 작성하고 배포까지 진행합니다.
