Gmail API 대량 발송 — 가능 여부

고객 등록 후 신규 정책 안내 메일을 일괄 발송하는 것은 기술적으로 가능합니다.
다만, Gmail 계정 유형에 따라 일일 발송 한도가 있습니다.

계정 유형 일일 발송 한도 BOAS 적용
일반 Gmail (@gmail.com) 500건 / 24시간 충분
Google Workspace 2,000건 / 24시간 충분
신규 계정 (14일 이내) 제한적 (20~100건) 주의
결론: 현재 suaappa@gmail.com 계정 기준으로 일 500건까지 발송 가능합니다.
BOAS 고객 규모 감안 시 충분한 수량이며, 관리자 대시보드에서 "정책 안내 일괄 발송" 기능을 구현할 수 있습니다.

구현 방식: 관리자가 고객 목록 선택 → 템플릿 선택 → 일괄 발송 (Worker에서 순차 발송, 초당 1건 제한)
